Über Phexor

Phexor ist ein Opensource File explorer welcher das Ziel hat eine coole Alternative zum
weit verbreiteten File Explorer von Windows darzustellen, 
Besonders liegt bei Phexor der Fokus auf die Benutzer Freundlichkeit und insbesondere
das eingehen auf Feedback und Verbesserungsvorschläge.
Phexor ist ein Projekt welches komplett Frei zugänglich ist und bleiben soll um 
unter anderem dem Ruf " Im Internet ist nichts umsonst, selbst die dinge Welche so Erscheinen" 
Entgegen zu wirken, Jedoch sind wir in unseren Mitteln Beschränkt das Projekt stetig mit Updates
zu versehen da wir nicht sonderlich viele Personen sind.
Das Projekt wurde durch Pascal Lösch (kallioderso) ins Leben gerufen und läuft auf Github 
unter einer GPLv3 Lizenz von Gnu

Vorschau

Hier können sie sich in einen kurzen Überblick verschaffen, dazu wie Phexor aussieht (Tutorial, Mainscreen, Einstellungen), je nach Version unterscheidet sich jedoch das aussehen teils stark

Tutorial

In unserem Kurzen aber Effektiven Tutorial werden 
alle Grundlegenden für das arbeiten mit Phexor
nötigen Funktionen erklärt, so das es Potentiell
jeder verstehen kann, und einen Guten Start ins arbeiten mit Phexor erhält 

Mainscreen

Unser Mainscreen ist der Ort in welchem
die User tendenziell die meiste zeit 
der Nutzung von Phexor verbringen werden,
hier können Ordner gesucht werden (Keine Ausblendungen), Dateien Geöffnet werden.
Also im Grunde können bisher mit kompletter Freiheit, Ordner durchstöbert werden (Weitere Features kommen zukünftig)

Einstellungen

Hier können unsere Nutzer alle Funktionen und
Optischen Einstellungen für ihr Arbeiten 
in Phexor einstellen (Jetziger stand nur Optische Dinge), weitere Features kommen mit der Zeit hinzu und Ermöglichen ein Freieres und Funktionsreicheres Arbeiten mit Phexor.

Wie wird das Projekt Finanziert?

Das Projekt hat an sich keine Einnahme quelle, weder Sponsoren, noch spenden oder verkaufseinnahmen , dies wird sich auch nicht ändern da das Projekt stehts Gratis bleiben soll und sich von anderen Opensourceprojekten abheben soll mit dem Verzicht bzw. der Ablehnung von Spenden

Wieso ist das Projekt in C# geschrieben?

Das Projekt ist im Grunde in C# geschrieben da dies die Sprache ist welche ich als erstes begann zu lernen, und vorhabe in meinem Späteren beruflichen leben praktisch anzuwenden

Welches Ziel Verfolgst du mit Phexor?

Mit Phexor Möchte ich im Grunde eine opensource File explorer alternative zum Explorer von Windows schaffen, welche jedoch so leicht zu verstehen ist das sie für jeden Nutzbar ist, gleichzeitig jedoch auch alle Tools bietet welche für ein gutes arbeiten nötig sind

Wieso gibt es Phexor bisher nur für windows?

Phexor gibt es bisher nur für Windows aus dem Grund, das ich erst einmal das Programm auf ein gewisses Niveau bringen will, und danach dann es für alle Betriebssysteme kompatibel machen will
